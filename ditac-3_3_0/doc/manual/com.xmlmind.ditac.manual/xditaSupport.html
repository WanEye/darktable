XDITA support

XMLmind DITA Converter fully supports XDITA , which is basically a very small subset of DITA XML, plus new audio and video elements .

Template of an XDITA topic (lwdita_templates/xdita_topic.dita ):
<?xml version="1.0" encoding="UTF-8"?>
<!DOCTYPE topic PUBLIC "-//OASIS//DTD LIGHTWEIGHT DITA Topic//EN"
"lw-topic.dtd">
<topic id="???">
  <title></title>
  <shortdesc></shortdesc>
  <body>
    <p></p>
  </body>
</topic>
Template of an XDITA map (lwdita_templates/xdita_map.dita ):
<?xml version="1.0" encoding="UTF-8"?>
<!DOCTYPE map PUBLIC "-//OASIS//DTD LIGHTWEIGHT DITA Map//EN"
"lw-map.dtd">
<map>
  <topicmeta>
    <navtitle></navtitle>
  </topicmeta>
  <topicref href="???">
    <topicref href="???"/>
  </topicref>
  <topicref href="???"/>
</map>
